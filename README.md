# Agentic-Hiring & Autonomous Placement Workflow
**ET GenAI Hackathon 2026 - Phase 2 Submission**

## 🚀 Overview
This project addresses **Problem Statement 2: Agentic AI for Autonomous Enterprise Workflows**. It is an end-to-end autonomous system designed to manage the enterprise hiring lifecycle, detect skill gaps, and self-correct through automated learning paths.

## 🧠 Multi-Agent Architecture
- **Intake Agent:** Parses complex resumes and profiles using Gemini 1.5 Flash.
- **Market Intelligence Agent:** Analyzes real-time industry demands to identify required tech-stacks.
- **Audit & Decision Agent:** Evaluates the "Readiness Score" and maintains an auditable trail of every decision.
- **Autonomous Recovery Agent:** If a candidate is not ready, this agent automatically triggers a personalized 15-day roadmap without human intervention.

## 🛠️ Tech Stack
- **Orchestration:** n8n (Agentic Workflows)
- **AI Models:** Gemini 1.5 Flash / Pro
- **Backend:** Python (FastAPI)
- **Environment:** Dockerized for enterprise scalability

## 📈 Business Impact
- **Efficiency:** 99% reduction in manual screening time.
- **Cost:** Estimated savings of ₹1,200 per candidate profile.
- **Optimization:** Increases the "hireable" talent pool by 1.5x through autonomous skill recovery.
